🗣

Install

# Clone and build
git clone https://github.com/augmentedmike/whisper-hotkey.git
cd whisper-hotkey
make install

# Run it
whisper-hotkey

The Whisper model (~142 MB) is downloaded automatically on first launch.

Why Whisper Hotkey?

Cost money per minute. Your audio leaves your machine.

Whisper Hotkey

Requirements

  • macOS 13 or later
  • Microphone permission
  • Accessibility permission (for typing into apps)
  • Xcode Command Line Tools (for building from source)
